# 🎧 Audoria – AI-Powered Educational Assistant for the Visually Impaired

Audoria is an AI-powered mobile educational application built to enable **visually impaired students** to study independently through a fully **voice-first learning experience**. The app transforms printed and digital learning materials into accessible audio-based lessons using OCR, Text-to-Speech, and AI-powered understanding features such as summarization, quizzes, and open Q&A.

Audoria removes learning barriers by eliminating visual dependency, simplifying interaction, and providing intelligent learning tools. It bridges communication between students and guardians through a **parent–child system** with insights and progress tracking, delivering an inclusive, scalable educational solution.

---

## 🔧 Features

### 🎙️ Voice-First Student Experience
- 100% hands-free navigation using voice commands
- No typing or visual interaction required
- Voice commands for capture, summarize, quiz, and AI chat

### 📷 OCR & Content Extraction
- Real-time camera OCR for printed materials
- Automatic text extraction from PDFs, images, and documents
- High-accuracy text recognition using Google ML Kit

### 🤖 AI-Powered Learning
- AI-generated summaries and key topics
- Automatic quiz generation from learning material
- Open AI chat for asking questions and clarification

### 👨‍👩‍👧 Parent–Child System
- Parents upload and assign study materials
- Monitor student progress and quiz results
- Performance insights and activity tracking
- QR-based child login (no credentials typing)

### 🔊 Accessibility & Feedback
- Text-to-Speech for all outputs
- Audio feedback for every action and result
- Simple, high-contrast UI optimized for accessibility

---

## 💡 Impact
- Enabled **independent learning** for visually impaired students
- Converted inaccessible educational materials into interactive audio lessons
- Reduced parental workload through automated progress insights
- Delivered a fully voice-navigable learning experience
- Built a scalable foundation for inclusive digital education
